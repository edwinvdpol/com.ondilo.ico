ICO övervakar din pool / spa.

Den analyserar din vattenkvalitet, hjälper dig med ditt underhåll och förutser förändringar i vädret.

Funktioner inkluderar:
- Läs redoxvärde
- Läs pH-värdet
- Läs salthaltvärde
- Läs totalt upplösta fasta ämnen
- Läs mottagen signalstyrka
- Utlösare tillgängliga för alla värden, inklusive batteri och temperatur
- Visar programversion, serienummer och volym på din pool / spa

Homey uppdaterar data automatiskt var 15: e minut.

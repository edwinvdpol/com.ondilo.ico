ICO vil overvåke bassenget / spaet ditt.

Den analyserer vannkvaliteten din, hjelper deg med vedlikeholdet og forventer endringer i været.

Funksjonene inkluderer:
- Les redoksverdi
- Les pH-verdien
- Les saltholdighetsverdien
- Les totalt oppløste faste stoffer
- Les mottatt signalstyrke
- Utløsere tilgjengelig for alle verdier, inkludert batteri og temperatur
- Viser programvareversjon, serienummer og volum på bassenget / spaet ditt

Homey vil automatisk oppdatere dataene hvert 15. minutt.
